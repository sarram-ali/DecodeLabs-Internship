# 📌 Project Overview

This project was completed as **Week 3** of my **Data Analytics Internship at Decode Labs**.

The objective was to perform **Exploratory Data Analysis (EDA)** using SQL by querying an e-commerce sales database and extracting meaningful business insights.

---

# 🎯 Internship Task

Analyze the sales database using SQL to:

- Retrieve business metrics
- Aggregate sales information
- Analyze customer purchasing patterns
- Generate business insights using SQL queries

---

# 📂 Dataset

The SQL database contains sales transaction records including:

- Order Information
- Products
- Quantity
- Payment Methods
- Total Revenue

---

# 🔍 SQL Analysis Performed

### ✔ Overall Business Metrics

- Total Orders
- Total Revenue
- Average Order Value

---

### ✔ Product Analysis

Calculated:

- Revenue by Product
- Quantity Sold by Product

---

### ✔ Payment Method Analysis

Analyzed customer payment preferences using SQL aggregation.

---

### ✔ Top Orders

Retrieved Top 10 highest-value orders based on TotalPrice.

---

# 💡 Business Insights

- Identified highest revenue generating products.
- Determined most frequently sold products.
- Calculated total business revenue.
- Compared payment methods.
- Found highest-value customer orders.

---
